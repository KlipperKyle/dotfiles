dotfiles
========

Miscellaneous personal config files.

Often times, I link people to various examples here.  Use at your own
risk.  If something is helpful, then great!  Feel free to use it.  If
something breaks your system, well sorry.

There is a lot of history here.  This repository has survived my
journey through my college years and several full-time engineering
jobs.  As a result, things are a little messy, so please set your
expectations accordingly.  The various configuration files and hacks
span several desktop environments and Linux distributions.  The status
of each configuration file varies from working to archival.
